# IPL Dataset Analysis Report

## Objective
To analyze an IPL players dataset using Python and Pandas.

## Dataset
The dataset contains:
- Player Name
- Team
- Runs
- Wickets
- Strike Rate
- Team Wins

## Analysis Performed
1. Highest Run Scorer
2. Most Wickets
3. Best Strike Rate
4. Team with Most Wins

## Results
- Highest Run Scorer: Suryakumar Yadav (899 Runs)
- Most Wickets: Mohammed Siraj (35 Wickets)
- Best Strike Rate: Matheesha Pathirana (188.13)
- Team with Most Wins: Gujarat Titans (14 Wins)

## Conclusion
The analysis successfully identified the top-performing player and team based on the dataset using Python and Pandas.
