# IPL Dataset Analysis

## Description

This project analyzes an IPL Players dataset created in CSV format using Python and the Pandas library. The program reads player statistics from a CSV file and identifies the top-performing players and team based on different performance metrics.

## Objectives

- Read IPL player data from a CSV file.
- Analyze player and team performance.
- Practice CSV data processing using Python and Pandas.

## Features

- Finds the Highest Run Scorer
- Finds the Player with the Most Wickets
- Finds the Best Strike Rate
- Finds the Team with the Most Wins

## Technologies Used

- Python
- Pandas
- CSV

## Project Files

- `ipl_players.csv` – IPL players dataset
- `ipl_analysis.py` – Python program for data analysis
- `README.md` – Project documentation
- `requirements.txt` – Project dependencies
- `Analysis_Report.md` – Analysis report

## Requirements

Install the required library:

```bash
pip install pandas
```

## How to Run

1. Open the project in PyCharm.
2. Ensure all project files are in the same folder.
3. Run `ipl_analysis.py`.
4. View the analysis results in the console.

## Sample Output

The program displays:

- Highest Run Scorer
- Most Wickets
- Best Strike Rate
- Team with Most Wins
