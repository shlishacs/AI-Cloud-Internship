# IPL Dataset Analysis

## Description

This project analyzes an IPL Players dataset stored in CSV format using **Python** and the **Pandas** library. It provides a menu-driven interface that allows users to explore player and team statistics, search for players, and analyze overall team performance.

## Objectives

* Read IPL player data from a CSV file.
* Analyze player and team performance using Pandas.
* Perform data filtering, grouping, and aggregation.
* Practice working with CSV datasets in Python.

## Features

* Display all player details.
* Display all teams and their members.
* Find the Highest Run Scorer.
* Find the Player with the Most Wickets.
* Find the Best Strike Rate.
* Search for a player by name.
* Calculate Average Runs by Team.
* Identify the Best All-Rounder.
* Display the Team with the Most Wins.
* Interactive menu-driven console application.

## Technologies Used

## How to Run

1. Open the project in **PyCharm** or **VS Code**.
2. Ensure all project files are in the same folder.
3. Install the required dependency:

   ```bash
   pip install pandas
   ```
4. Run `ipl_analysis.py`.
5. Select an option from the menu to view the desired analysis.
