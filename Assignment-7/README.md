# IPL Dataset Analysis

## Description
This project analyzes an IPL players dataset created in CSV format using Python and Pandas.

## Features
- Finds the Highest Run Scorer
- Finds the Player with the Most Wickets
- Finds the Best Strike Rate
- Finds the Team with the Most Wins

## Technologies Used
- Python
- Pandas
- CSV

## Files
- ipl_players.csv
- ipl_analysis.py
- README.md
- requirements.txt

## How to Run
1. Open the project in PyCharm.
2. Ensure `ipl_players.csv` is in the project folder.
3. Run `ipl_analysis.py`.

## Output
The program displays:
- Highest Run Scorer
- Most Wickets
- Best Strike Rate
- Team with Most Wins
