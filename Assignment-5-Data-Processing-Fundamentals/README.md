# Data Processing Fundamentals

## Objective
This project demonstrates basic data processing using Python and Pandas.

## Dataset
Student Dataset (CSV)

## Tasks Performed
- Read CSV file
- Removed null values
- Counted total records
- Generated summary statistics
- Saved cleaned dataset

## Technologies Used
- Python
- Pandas

## How to Run

1. Install pandas

```
pip install pandas
```

2. Run the program

```
python data_processing.py
```
